
# Music Player
Music Player with simple user interface and functionality to take time laps with click on progress bar.




## Features
- Play push button
- Responsive progress var in wave form
- Inbuilt timer 
- Dark theme 


## Tech-Stacks 
- HTML
- CSS
- JavaScript
- Google Fonts 
- fontawesome 4.7
